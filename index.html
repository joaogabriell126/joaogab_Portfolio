Meus projetos

Sispar - Wilsons Sons

O SISPAR é um projeto visual de um sistema de reembolsos desenvolvido para a Wilson Sons. Criado com a framework React.js, o projeto incorpora conceitos essenciais do desenvolvimento web, como sistemas de rotas, formulários e tabelas, proporcionando uma interface interativa e funcional.

Netflix - Clone

Um projeto de desenvolvimento com o objetivo de reproduzir a página de login da Netflix, uma das maiores plataformas de streaming do mundo. O foco está na fidelidade visual e na aplicação de boas práticas de desenvolvimento web.

Calculadora - IMC

Um projeto simples, porém muito útil: uma calculadora responsiva que demonstra de forma clara as práticas de JavaScript, integradas com HTML e CSS. Seu principal objetivo é calcular e exibir o resultado com base no peso e na altura do usuário.
